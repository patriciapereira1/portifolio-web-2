Experiências

Jogo Secreto

O "Jogo Secreto" é um projeto interativo desenvolvido com HTML, CSS e JavaScript, que desafia os jogadores a adivinharem um número secreto. O jogo oferece uma experiência divertida e educativa, permitindo que os jogadores testem suas habilidades de adivinhação. A cada tentativa, o jogo informa se o número digitado é maior ou menor que o número secreto. O jogo acompanha e exibe a quantidade de tentativas realizadas pelo jogador. Quando o jogador acerta o número, uma mensagem é exibida, parabenizando e mostrando o total de tentativas.
